# Vastra

Vastra is a professional CRM and commerce management platform developed by Pranathi. It is designed to provide seamless business management with a modern, responsive, and dark-themed user interface.

## Features
- **Dashboard Overview:** Monitor your business growth and metrics at a glance.
- **Customer Management:** Efficiently manage customer relationships and interactions.
- **Product Management:** Track inventory, manage products, and categorize offerings.
- **Order Tracking:** Keep track of ongoing and past orders.
- **Responsive Layout:** Works smoothly on desktop and mobile devices.
- **Dark Mode:** A sleek and modern dark UI built for eye comfort.

## Development

Developed using modern web technologies:
- React
- TypeScript
- Vite
- Tailwind CSS

## Deployment

Follow the standard Vite deployment instructions to build and serve the application:
```bash
npm run build
npm start
```
