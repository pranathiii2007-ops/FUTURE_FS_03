# Vastra: Premium Indian Boutique & Commerce Platform

![Vastra Heritage](https://images.unsplash.com/photo-1599839619722-39751411ea63?auto=format&fit=crop&q=80&w=1200)

**Vastra** is an elegant, full-featured modern web storefront crafted exclusively for premium Indian ethnic wear and boutique fashion. Designed and developed by **Pranathi**, the application combines a stunning, luxury aesthetic with seamless ecommerce functionality.

## 🌟 The Vision

The goal of Vastra was to move away from standard, generic e-commerce templates and bring a true sense of heritage and luxury to the digital shopping experience. 

The application utilizes a custom-designed **Deep Burgundy and Rich Gold** color palette set against a soft, breathable Ivory background. This bespoke theme captures the essence of Indian bridal and festive wear, providing users with a highly immersive and visually satisfying shopping journey.

## ✨ Key Features

- **Storefront & Catalog:** A beautifully curated shop section displaying handpicked ethnic wear, jewelry, and accessories with detailed, high-resolution product cards.
- **Product Details:** Clean, minimalist product pages focusing on high-quality imagery, descriptions, and dynamic size selections.
- **Shopping Bag & Checkout Flow:** A fully functional, state-managed shopping cart that calculates subtotals, applies promo codes (e.g., *WELCOME10*), and seamlessly transitions users to checkout.
- **Personal Styling Appointments:** A dedicated booking system allowing users to schedule in-store, one-on-one styling consultations at our Vishakapatnam boutique.
- **Responsive Architecture:** Perfectly optimized for mobile, tablet, and desktop viewing without compromising the luxury aesthetic.
- **Performance Optimized:** Built with Vite and React for instantaneous page loads, micro-animations, and fluid routing.

## 🛠️ Technology Stack

Vastra is powered by a modern, high-performance frontend architecture:

* **Framework:** [React 18](https://react.dev/)
* **Build Tool:** [Vite](https://vitejs.dev/) - Lightning fast Cold Starts and Hot Module Replacement (HMR).
* **Language:** [TypeScript](https://www.typescriptlang.org/) - Ensuring type safety and robust code architecture.
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) & [Shadcn UI](https://ui.shadcn.com/) - Utilizing bespoke CSS variables for dynamic, elegant theme integration.
* **Icons:** [Lucide React](https://lucide.dev/)
* **Routing:** [React Router v6](https://reactrouter.com/)

## 🚀 Getting Started

To run Vastra on your local machine, follow these simple steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (version 18 or above) installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/pranathiii2007-ops/FUTURE_FS_03.git
   cd FUTURE_FS_03
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
   The application will be running locally at `http://localhost:8080/`.

4. **Build for Production:**
   ```bash
   npm run build
   ```
